Ecommerce Website

This ecommerce website is meticulously crafted using a stack that includes Svelte for a responsive and dynamic frontend, Node.js to handle server-side operations, and a blend of HTML, JavaScript, and Tailwind CSS for a sleek and intuitive user interface. The use of Svelte ensures efficient state management and seamless interaction, while Node.js supports robust backend functionalities, facilitating secure transactions and data management. The combination of Tailwind CSS and custom styling results in a visually appealing and modern design. The website is designed to provide users with a seamless shopping experience, incorporating features like product listings, a shopping cart, and a secure checkout process. With its elegant design and smooth functionality, this ecommerce site exemplifies the synergy of these technologies to create a compelling online shopping platform.

Todo App

This todo app is a well-rounded web application crafted with a combination of Svelte, Node.js, HTML, JavaScript, and Tailwind CSS. The frontend, developed with Svelte, ensures a highly reactive and dynamic user interface, while the backend, powered by Node.js, manages the server-side logic. MongoDB serves as the NoSQL database for efficient storage and retrieval of todo data. The seamless deployment on Vercel ensures accessibility to users, and Git enables version control. With essential features like CRUD operations for todos and potential user authentication, the app showcases a responsive design for optimal usability across various devices. The incorporation of CORS configuration ensures secure API interactions. Overall, this todo app represents a harmonious integration of technologies to deliver a functional and user-friendly experience.
